# 🚀 Whenevr Clone – Premium SaaS Design Website

A pixel-perfect, fully responsive clone of a modern Framer-style SaaS website built using **Next.js, Tailwind CSS, and Framer Motion**.

This project focuses on **UI precision, smooth animations, and real-world interaction flows** like booking systems and multi-page navigation.

## ✨ Features

* ⚡ Pixel-perfect UI inspired by Framer design
* 🎨 Modern typography (Inter + Serif combination)
* 💎 Glassmorphism & soft shadow design system
* 📱 Fully responsive (mobile, tablet, desktop)
* 🎞️ Smooth animations using Framer Motion
* 🧠 Interactive booking flow (Calendar → Time → Form → Confirmation)
* 🔁 Auto-scrolling testimonial slider
* 🎯 Hover interactions and micro-animations
* 📄 Multiple pages:

  * Home
  * Thank You
  * Privacy Policy
  * Terms of Service
  * 404 Page

## 🚀 Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/pratikdas018/whenevr.git
cd whenevr
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```

Open 👉 http://localhost:3000
